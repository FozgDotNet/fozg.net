## fozg.net Landing page 
A simple Landing page builded with bootstrap & sass.

### `Install`
```bash
git clone https://github.com/fozg/fozg.net.git  
cd fozg.net
npm install

# install gulp global
npm install -g gulp
```
### `npm start`
Runs the app in development mode.
Open http://localhost:3000 to view it in the browser.
